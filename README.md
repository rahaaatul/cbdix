# CBDIX

Check multiple BDIX servers simultaneously.

## Installation

> [!TIP]
> You can also use `pip` as alternative.

```bash
pipx install git+https://github.com/rahaaatul/cbdix
```

## Usage

> [!TIP]
> You can also use `bdix` as alternative.

```bash
cbdix run --help
```
